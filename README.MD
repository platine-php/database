### Requirements 
- **PHP >= 7.4**, **PHP 8** 

### Installation
#### Using composer (recommended)
```bash
composer require platine-php/database
```
### Current supported drivers

- MySQL/MariaDB
- SQLite
- PostgreSQL
- Oracle
- Microsoft SQL Server
